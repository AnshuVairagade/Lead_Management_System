# Lead Management System

A lead management system built with React, Django, and SQLAlchemy for creating, updating, and tracking leads through a simple and clean dashboard.

## Features

- Add new leads with essential contact details
- Update existing leads through an intuitive form
- Delete leads from the dashboard
- Display leads in a clean table layout
- Track last updated time for every lead
- Secure backend API for lead management

## Tech Stack

**Frontend**
- React  
- Axios  
- Tailwind or CSS modules  

**Backend**
- Django  
- SQLAlchemy  
- REST API architecture  


## How It Works

The frontend communicates with the backend using REST endpoints. Lead data is stored and managed using SQLAlchemy models. Users can create, edit, and delete leads through the dashboard.

## Setup Instructions

### Backend Setup

1. Clone the repository :  ```git clone <your repo link>```

2. Navigate to the backend folder  : ```cd backend```

3. Install dependencies  : ```pip install -r requirements.txt```

3. Install dependencies  : ```python manage.py runserver```


### Frontend Setup

1. Navigate to the frontend folder : ```cd frontend```


2. Install dependencies : ```npm install```

3. Start the React app : ```npm start```
